## 功能概述<br>
本项目为实现了多功能的综合计算器，适用对象为精致or想变精致的猪猪女孩/男孩，主要功能如下：<br>
1. 基础计算器功能：实现了加减乘除四则混合运算、进制换算以及逻辑位运算；<br>
2. 精致功能：包括账单统计和化妆品单位价格计算。<br>
